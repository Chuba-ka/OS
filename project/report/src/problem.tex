\section{Условие задачи}

\textbf{Цель работы:} разработать клиент-серверную систему передачи мгновенных сообщений с использованием разделяемой памяти.

\textbf{Задание:}

Реализовать клиент-серверное приложение со следующим обязательным функционалом:
\begin{itemize}
  \item Клиент может присоединиться к серверу, указав логин.
  \item Клиент может отправить личное сообщение другому клиенту по его логину.
  \item Клиент получает сообщения от других пользователей в реальном времени.
  \item На сервере должна храниться история всех переписок.
  \item Должен быть реализован поиск по истории переписки.
  \item Связь между сервером и клиентами должна быть реализована исключительно с помощью механизма отображения файлов в память (memory map, POSIX shared memory).
\end{itemize}
